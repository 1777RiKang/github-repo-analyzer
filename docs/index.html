GitHub Repo Analyzer

10 维度数据分析 · 10 张可视化图表 · 交互式 Web · PDF 报告导出

关于本项目

输入任意 GitHub 用户名（如 torvalds、microsoft），一键采集所有公开仓库数据，进行 10 维度深度分析，生成交互式图表和 PDF 报告。

Tech Stack: Python · Streamlit · pandas · matplotlib · wordcloud

快速开始

git clone https://github.com/1777RiKang/github-repo-analyzer.git

pip install -r requirements.txt

python run_web.py

浏览器自动打开 http://localhost:8501
